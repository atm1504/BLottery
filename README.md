# Lottery
A blockchain app to create a lottery draw system.

## How to start
* npm install

## How to Test
* npm run test


## How to deploy
* First add a config.js file with the ```RINKBEY_URL``` and ```MNEMONIC```.
* Now execute ```node deploy.js```

## Frontend Part:
https://github.com/atm1504/BLottery-Frontend
